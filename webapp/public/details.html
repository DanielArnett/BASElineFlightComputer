Summary, Sensor Integration, Logging, Feedback BASEline is a mobile app designed to improve human performance by giving athletes real-time audible feedback. Phone sensors provide real-time data on speed, position, orientation, etc.

  1. Requirements

    To obtain accurate altitude data, some devices have built-in barometric pressure sensors. If not present, the software will fallback to GPS-based altitude; however, GPS ALTITUDE IS NOT RELIABLE. Devices known to have a barometric altimeter:
    • Samsung Galaxy Nexus, Galaxy SIII, Galaxy Note
    • Motorola Droid 4, Xoom 1 & 2
    • Huawei Ascend D
    • Sony Xperia Active
    Android 3.0 or greater is required.

  2. Visual Altimeter and Flight Computer

    The user's altitude, and other data is displayed by the app through the various tabs. Altitude is always measured Above Ground Level (AGL). Ground level is calibrated when the app is started such that the altitude always starts at zero. To change the ground level, go to Settings, and select "Adjust Ground Level". Drag up and down to change the ground level.

  3. Audible

    The audible lets you to create custom audio events based on sensor readings. Each audible event consists of 3 parts:
    • Trigger - The event fires when all the trigger conditions are satisfied.
    • Sound - This is the sound sample that will play when triggered.
    • Modifiers - This allows you to modify the sound based on phone sensors.

  4. Navigation

    Navigation is achieved by setting a "home" location, and then using audible events to indicate distance and bearing to the home location.
    • "Home" location is set on first GPS lock, or set by user in Map tab.
    • Audible can be programmed to use "Distance Home" and "Bearing Home" sensors.
    • Bearing Home is the angle difference between the direction of travel and the bearing to the home location.

  5. Jump Log

    The Jump Log records all sensor readings while the app is running.
    • Exporting requires pulling lots of sensor data from the database, and can be very slow!
    • Files are exported to the phone's "Download" directory (/sdcard/Download/).
    • To access the "Download" directory, connect the phone to the computer using the USB cable. The phone must be set to connect as a "Media Device" (MTP), not as a Camera (PTP).
